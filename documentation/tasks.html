Developer Observatory - Documentation

Preconditions:

  • Python 3
  • Flask

We created a tool to help you generate the task files. A notebook file always has to follow the scheme:
  • Introduction (1 markdown cell)
  • any number of tasks (1 markdown cell + 1 code cell)
If you want to change this, you'll have to manually modify the JavaScript in instance_server/template/custom.js If you are using a completly different set of tasks, for example, symetric vs asymetric encryption, then make sure that both files contain the same amount of cells+tasks. Incase that you have a different amount of tasks, you can also modify instance_server/template/custom.js to return the correct amount depending on the amount of cells in the document ($('.cell').length)

1) Start the notebook generator

Install the required dependencies with pip3 install -r requirements.txt, run python3 notebookgen.py and open http://localhost:5000 in your webbrowser.

2) Generate Cells

Switch to the Cells tab and click on "New Cell". You may now create Markdown (text only) cells and Code cells. A task will always need a Markdown cell followed by a Code cell except for the introduction which consists of a single Markdown cell. Give the Code cells a metadata field called "tasknum" with a number to identify the task later on.

3) Generate Tasks

Switch to the Tasks tab and click on "New Task" and select the appropriate cells for the task.

4) Create Conditions

Switch to the Conditions tab and create conditions which are used to replace placeholders in cells.

5) Generate Notebooks

The last step is to generate the notebooks. Switch to the Notebooks tab and create a new notebook template. You can now select which tasks should be included in this notebook file. You can also select if certain tasks should be at a fixed position. The intro should always be fixed to the first position, followed by tasks in latin squares order.
